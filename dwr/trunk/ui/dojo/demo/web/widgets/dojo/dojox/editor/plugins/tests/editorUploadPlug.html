Click the upload icon to open a system dialog. Note that this will need to call a server script for the upload to work.

Test Disclaimers:
A PHP test file is included. PHP will need to be enabled.
If you are viewing this in on the web in the dojo archives, it will not work - uploading is not enabled.
Folder permissions will need to be set to system level for uploads to work on your local system.
This code uses the GD library (to get image sizes), that sometimes is not pre-installed in a standard PHP build.
