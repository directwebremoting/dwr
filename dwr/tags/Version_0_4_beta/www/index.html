DWR (Direct Web Remoting)

DWR gives you:

  • a very easy way to call Java code directly from Javascript including
  • automatic conversion of numbers, strings and dates, conversion of Javascript arrays into Java Collections and lists and back, conversion of Javascript objects into Java beans and maps and back.
  • a jump-start to being able to create GMail type interactivity.

DWR does not:

  • attempt to be a servlet framework
  • demand that you create complex configuration files or write lots of boilerplate Action classes.

Steps to getting DWR running in your web app:

You can't run much of a demo site using the java.net servers so there is a main DWR website that includes demos of how it works along with lots of documentation.

To help get started there is an introduction on how to add the jar file to your webapp, and then more detail on how to configure DWR and a detailed configuration reference. There is further documentation linked from the main DWR website.
