Anti-spam mailto: Link Creator.

You can put mailto: links in HTML to allow people to send you email with a simple click. In practice this allows spammers to harvest addresses easily. The solution is to use a simple script to generate mailto: links.

The following paragraph is an example:

Contact the helpdesk using:

If you check the source, you'll see that the address is hidden well enough that an address harvesting robot will miss it.
