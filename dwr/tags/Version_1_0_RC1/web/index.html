DWR is NOT working properly

This is a fairly advanced page that tests DWR. If you can see this message then you should check the following:

  • Does the entry in the address bar begin 'file:'?
    If so you need to view this website through a web server to get the benefit of DWR.
  • Can you see the test pages?
    If not it is likely that DWR has not deployed in your web server properly; Check the web server console and log files for error messages
  • If you can see the test pages then DWR is loaded but is something is preventing DWR working. Check you browser Javascript console if possible, or see the main DWR website for more information.
