Direct Web Remoting

DWR allows Javascript in a browser to interact with Java on a server and helps you manipulate web pages with the results.

DWR is Easy Ajax for Java

 

Our stable release is DWR version 1.1.3. You can see the release notes, download it now and use it in your website in minutes.

We are developing DWR version 2.0, and recently announced 'Reverse Ajax' which allows Java on the server to asynchronously send JavaScript to the client. See the release notes for more information, or download it now.

   

DWR is Open Source, available under the Apache Software License v2, so it is easy to Ajax enable your website.

DWR documentation and tutorials are available in languages other than English. We are collecting non-English DWR pages.
