DWR stands for Direct Web Remoting. It is Easy Ajax for Java. It allows Javascript in a browser to interact with Java on a server and helps you manipulate web pages with the results. DWR makes it easy for you to Ajax enable your website.

Our stable release is DWR version 1.1. You can see the release notes, download it now and use it in your website in minutes.

We are developing DWR version 2.0, and recently announced 'Reverse Ajax' which allows Java on the server to asynchronously send Javascript to the client. See the release notes for milestone 1, or download 2.0m1 now.

DWR is freely available as open source software (ASL version 2.0). It is straight forward to implement with extensive examples and tutorials. Incorporating it into existing sites is simple as it readily integrates with the most commonly used Java frameworks.
