Direct Web Remoting

DWR allows Javascript in a browser to interact with Java on a server and helps you manipulate web pages with the results.

DWR is Easy Ajax for Java

 

Our stable release is DWR version 1.1. You can see the release notes, download it now and use it in your website in minutes.

We are developing DWR version 2.0, and recently announced 'Reverse Ajax' which allows Java on the server to asynchronously send Javascript to the client. See the release notes for milestone 1, or download it now.

 

   

 

DWR makes it easy for you to Ajax enable your website, and it's open source (ASL v2) so you can start right away.
